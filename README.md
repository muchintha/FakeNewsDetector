# Fake News Detector

A Python project to classify news headlines as FAKE or REAL using a machine learning model.

## Features
- Predicts if a news headline is REAL or FAKE
- Color-coded GUI (green for REAL, red for FAKE)
- Displays confidence percentage for each prediction
- Stores prediction history for review

## Technologies Used
- Python
- Tkinter for GUI
- Pandas for data handling
- Scikit-learn (Naive Bayes, TF-IDF)

## How to Run
1. Install required libraries: 2. Place Fake.csv and True.csv in the project folder
3. Run:
4. Enter a news headline in the GUI and click "Predict"

## Dataset
Uses publicly available fake and real news datasets.
